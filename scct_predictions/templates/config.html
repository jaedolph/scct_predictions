Setup Instructions

1. Create a new application at: https://dev.twitch.tv/console/apps/create
2. Set "Name" to whatever you want e.g. "scct_predictions"
3. Add an "OAuth Redirect URL" to http://localhost:{{ app_port }}/configure/confirm
4. Set "Category" to "Application Integration"
5. Set "Client" to "Confidential"
6. Click the "I'm not a robot" verification and click "Create"
7. Click "manage" on the application you have created
8. Click "New Secret" to generate a "Client Secret"
9. Copy/paste the "Client ID" and "Client Secret" in to the form below
10. Fill in any other fields and click "Submit"
