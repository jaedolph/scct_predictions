To setup Stream Deck

1. In the Stream Deck desktop application, create a new folder called "Predictions"
2. Add a System --> Website action, called "Create"
2. Set the url to "http://localhost:{{app_port}}/predictions/create"
3. Ensure "GET request in background" is ticked
4. (OPTIONAL) You can create additional actions at these endpoints:
    (a) Lock the prediction: http://localhost:{{app_port}}/predictions/lock
    (b) Payout the prediction: http://localhost:{{app_port}}/predictions/payout
    (c) Cancel the prediction: http://localhost:{{app_port}}/predictions/cancel

To start a prediction

1. Ensure SCCT is pre-filled with the current match details and the match is selected
2. Press the "Create" button on the stream deck

To lock a prediction

1. Press the "Lock" button on the stream deck

To pay out a prediction

1. Ensure the match is updated in SCCT
2. Press the "Payout" button on the stream deck
3. The tool should automatically payout the correct result

To cancel a prediction

1. Press the "Cancel" button on the stream deck
